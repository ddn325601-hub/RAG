## 3. 不应提交内容

以下内容不进入公开仓库，也不建议放入比赛提交包：

- 真实 `DASHSCOPE_API_KEY`。
- 真实 `CONTEST_API_TOKEN` 或任何长期有效访问凭证。
- 服务器 SSH 私钥、`.env` 文件、个人账号凭证。
- `target/`、`*.class`、`*.jar`、`*.tar.gz` 等构建产物。
- `uploads/`、`volumes/`、日志等运行数据。
- 个人学习材料、课程材料、面试复习材料等与项目交付无关的文件。
- 带有非正式来源说明的路径或材料。

## 4. 最终验证命令

本地编译验证：

```powershell
mvn -gs maven-settings-central.xml -s maven-settings-central.xml -DskipTests compile
```

敏感内容扫描：

```powershell
rg -n "sk-ws-|dev-contest-token|DASHSCOPE_API_KEY=.*sk|CONTEST_API_TOKEN=.*[A-Za-z0-9_-]{16,}" .
```

云端接口验证：

```powershell
Invoke-RestMethod "http://121.40.90.107/api/knowledge/files"

$q = [System.Uri]::EscapeDataString("支付订单超时怎么办")
Invoke-RestMethod "http://121.40.90.107/api/rag/search?q=$q&topK=3"
```

比赛接口验证需要单独设置 Token：

```powershell
$env:CONTEST_API_TOKEN = "<your-contest-token>"
.\contest-deliverables\api-examples\smoke-test.ps1 -BaseUrl "http://121.40.90.107" -Token $env:CONTEST_API_TOKEN
```
