# 接口说明

## 1. 比赛问答接口

### 请求

- Method：`POST`
- 本地 URL：`http://127.0.0.1:9900/chat`
- 公网 URL：`http://121.40.90.107/chat`
- Header：
  - `Authorization: Bearer <CONTEST_API_TOKEN>`
  - `Content-Type: application/json; charset=utf-8`

### 请求体

```json
{
  "question": "请根据内部知识库回答：支付订单超时应该怎么排查？",
  "images": [],
  "session_id": "demo-session",
  "stream": false
}
```

| 字段 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| `question` | string | 是 | 用户问题 |
| `images` | string[] | 否 | Base64 图片或 `data:image/...;base64,...` |
| `session_id` | string | 否 | 会话 ID，不传则自动生成 |
| `stream` | boolean | 否 | 当前比赛接口返回非流式结果，建议传 `false` |

### 成功响应

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "answer": "智能体回答内容",
    "session_id": "demo-session",
    "timestamp": 1781674766505
  }
}
```

### 错误响应

| 场景 | HTTP 状态 | 响应 |
| --- | --- | --- |
| 未带 Token 或 Token 错误 | 401 | `{"code":401,"msg":"unauthorized"}` |
| 问题为空 | 400 | `{"code":400,"msg":"question cannot be blank"}` |
| 图片过大或格式异常 | 400 | 返回具体校验错误 |
| 模型或服务异常 | 500 | `{"code":500,"msg":"chat failed: ..."}` |

### PowerShell 示例

```powershell
$baseUrl = "http://121.40.90.107"
$token = $env:CONTEST_API_TOKEN

$body = @{
  question = "支付订单超时应该怎么排查？"
  session_id = "demo-session"
  stream = $false
} | ConvertTo-Json -Depth 5

Invoke-RestMethod `
  -Method Post `
  -Uri "$baseUrl/chat" `
  -Headers @{ Authorization = "Bearer $token" } `
  -ContentType "application/json; charset=utf-8" `
  -Body $body
```

## 2. 知识库上传接口

### 请求

- Method：`POST`
- 本地 URL：`http://127.0.0.1:9900/api/upload`
- 公网 URL：`http://121.40.90.107/api/upload`
- Content-Type：`multipart/form-data`
- 参数：`file`

支持文件类型：`.txt`、`.md`

### 示例

```powershell
curl.exe -F "file=@C:\path\to\payment_order_runbook.md" http://121.40.90.107/api/upload
```

### 成功响应

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "fileName": "payment_order_runbook.md",
    "filePath": "uploads/payment_order_runbook.md",
    "fileSize": 1917
  }
}
```

## 3. RAG 召回调试接口

该接口用于学习、调试和验收，方便查看某个问题实际命中了哪些知识库片段。

### 请求

- Method：`GET`
- 本地 URL：`http://127.0.0.1:9900/api/rag/search?q=问题&topK=3`
- 公网 URL：`http://121.40.90.107/api/rag/search?q=问题&topK=3`

### 示例

```powershell
$q = [System.Uri]::EscapeDataString("支付订单超时怎么办")
Invoke-RestMethod "http://121.40.90.107/api/rag/search?q=$q&topK=3"
```

### 响应字段

| 字段 | 说明 |
| --- | --- |
| `id` | 向量片段 ID |
| `content` | 命中的知识库内容 |
| `score` | L2 距离，越小越相似 |
| `metadata` | 来源文件、标题、分片序号等 |

## 4. 知识库管理接口

### 请求

- Method：`GET`
- 本地 URL：`http://127.0.0.1:9900/api/knowledge/files`
- 公网 URL：`http://121.40.90.107/api/knowledge/files`

### 说明

该接口为前端“知识库”侧边栏提供文件列表，返回已上传资料的文件名、大小、路径和更新时间。

## 5. Milvus 检查接口

### 请求

- Method：`GET`
- 本地 URL：`http://127.0.0.1:9900/milvus/health`

公网环境下该接口默认也可通过 `http://121.40.90.107/milvus/health` 访问，用于演示验收；正式生产环境建议限制访问。

### 成功响应

```json
{
  "collections": ["biz"],
  "message": "ok"
}
```
