# 演示视频录制脚本

本文件记录当前产品功能演示视频的录制流程与复现命令。最终视频文件为：

```text
contest-deliverables/OnCallAgent_Function_Demo.mp4
```

视频为无音频的 1080p 产品实操素材，后续可继续与答辩 PPT、旁白或字幕进行剪辑组合。

## 1. 视频内容

| 时间段 | 画面内容 | 覆盖能力 |
| --- | --- | --- |
| 0:00-0:40 | 打开系统首页，依次展示左侧智能问答、知识库管理、接口调试入口 | 产品首页与系统结构 |
| 0:40-1:10 | 打开知识库管理面板，展示 7 份已入库资料、上传资料入口和入库状态 | 知识库管理、资料投喂 |
| 1:10-2:40 | 输入“支付订单超时如何排查？”，等待系统回答并展示引用来源 | RAG 问答、TopK=3 来源追溯 |
| 2:40-3:10 | 打开接口调试页面，分别执行无 Token 请求和带 Token 请求 | `/chat` 接口、Token 鉴权 |

## 2. 启动命令

在项目根目录启动本地服务：

```powershell
$env:CONTEST_API_TOKEN = "demo-recording-token"
mvn -gs maven-settings-central.xml -s maven-settings-central.xml spring-boot:run "-Dspring-boot.run.profiles=local" "-Dspring-boot.run.arguments=--server.port=9910"
```

浏览器访问地址：

```text
http://127.0.0.1:9910/
```

## 3. 录制命令

自动录制脚本：

```text
scripts/record_function_demo.py
```

执行命令：

```powershell
python scripts\record_function_demo.py --url http://127.0.0.1:9910/ --token demo-recording-token --output contest-deliverables\OnCallAgent_Function_Demo.mp4
```

脚本会启动一个临时 Chrome 实例，通过页面真实点击、输入、上传和接口请求完成录制。录制过程中不会显示 IDE、终端、开发者工具或真实密钥。
